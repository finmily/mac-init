### OSX Install

+ 简化dock(最底部的条)的内容(去除不常用的)，右键-选项-从dock中移除
+ 在系统偏好设置-安全性与隐私-通用中设置 ***任何来源***
+ 打开终端，输入git -v，如果弹出提示，选择安装 command line developer tool
